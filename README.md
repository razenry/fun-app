# Aplikasi Cek Kepribadian + Filter Kata Kasar

Aplikasi ini dibuat untuk keperluan edukasi dan hiburan di sekolah/kampus.  
Fitur filter kata kasar digunakan untuk menjaga konten tetap sopan dan positif.

File `ib/badwords.ts` berisi daftar kata yang difilter untuk keperluan moderasi konten.  
Bukan untuk tujuan negatif atau hate speech.
